# 🔍 Scanner TCP

Scanner de portas TCP desenvolvido em Python para identificação de portas abertas em um host.

## 🚀 Funcionalidades

- Verificação de portas TCP
- Timeout configurável
- Identificação de portas abertas
- Uso de sockets em Python

## 🛠️ Tecnologias

- Python
- Socket
- Linux

## ▶️ Execução

```bash
python3 scanner.py
```

Projeto desenvolvido para fins educacionais e laboratórios autorizados.
